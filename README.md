# Credit-Card-Financial-Analytics-Dashboard

Tools Used:
* Power BI
* SQL
* Excel

Key Metrics:
* Revenue: 55M
* Interest Earned: 7.84M
* Transaction Count: 656K

Business Insights:
* Blue Card customers generated the highest revenue.
* Bills and Entertainment contributed the largest transaction revenue.
* Graduate customers represented the highest revenue segment.
* Swipe transactions accounted for the majority of revenue.
